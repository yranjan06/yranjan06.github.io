# Ranjan's Portfolio

A modern, responsive portfolio website showcasing my work as a Data Engineer and Platform Developer.

**Live**: [yranjan06.github.io](https://yranjan06.github.io/)

---

## Features

- Modern dark theme with smooth animations
- Fully responsive design
- Project showcase with detailed pages
- Blog section
- Discussion system (Giscus-powered)
- Single Page Application architecture

---

## Tech Stack

- Vanilla JavaScript, HTML5, CSS3
- SASS for styling
- Webpack for bundling
- GitHub Pages for deployment

---

## Getting Started

```bash
# Clone
git clone https://github.com/yranjan06/yranjan06.github.io.git
cd yranjan06.github.io

# Install
npm install

# Run
npm run serve

# Build
npm run build
```

## Enabling Discussions

Project discussions are powered by [Giscus](https://giscus.app). To enable:

1. Enable Discussions in your GitHub repo settings
2. Generate config at giscus.app
3. Update values in `src/app/consts/giscusConfig.js`
